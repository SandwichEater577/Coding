# 🌐 FrontEnd

School coursework and exercises for front-end web development — HTML, CSS, and JavaScript.

## 📁 Sections

### S1 — HTML & CSS

Fundamentals of HTML structure and CSS styling.

### S2 — HTML, CSS & RWR

Responsive web design, layouts, and advanced HTML/CSS patterns.

### S3 — JavaScript

JavaScript programming exercises and exam prep.

- **Przyklady zadan na sprawdzian** — Practice problems with automated Jest tests
  - 6 exercises (`zadanie1.js` – `zadanie6.js`) with matching test files
  - Run exercises: `npm run z1`, `npm run z2`, etc.
  - Run tests: `npm run t1`, `npm run t2`, etc.
