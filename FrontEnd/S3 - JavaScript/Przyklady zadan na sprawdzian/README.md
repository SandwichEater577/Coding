# 📝 JavaScript Exam Prep

Practice tasks for a JavaScript exam — 6 exercises covering arrays, objects, DOM manipulation, and algorithmic thinking.

## ▶️ Running Exercises

```bash
npm install
npm run z1   # Run exercise 1
npm run z2   # Run exercise 2
# ... up to z6
```

## 🧪 Running Tests

```bash
npm run t1   # Test exercise 1
npm run t2   # Test exercise 2
# ... up to t6
```

## 📁 Files

| File                          | Description                       |
| ----------------------------- | --------------------------------- |
| `zadanie1.js` – `zadanie6.js` | Exercise implementations          |
| `tests/`                      | Jest test files for each exercise |
| `jest.config.js`              | Test runner configuration         |

> **Tip:** Use Google Chrome for the best experience.
