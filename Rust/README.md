# 🦀 Rust

Rust programming exercises organized by difficulty level.

## 📁 Sections

### S1 — Podstawy Rusta (Rust Basics)

10 exercises (`Cw1.rs` – `Cw10.rs`) covering:

- Printing and formatting (`println!`, `{}` placeholders)
- Variables, types, and mutability
- Control flow (`if`, `match`, loops)
- Functions and basic I/O

### S2 — Intermediate Rust

10 exercises (`Cw1.rs` – `Cw10.rs`) covering:

- Ownership and borrowing
- Structs, enums, and pattern matching
- Error handling (`Result`, `Option`)
- Collections, iterators, and closures

## ▶️ Running an Exercise

```bash
rustc Cw1.rs -o Cw1.exe && ./Cw1.exe
```

Or use `cargo` if set up as a project.
