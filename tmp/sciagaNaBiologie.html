🧬 BIOLOGIA: REPOZYTORIUM

Cytologia (Komórka)

  • Jądro: Centrum sterowania, zawiera DNA.
  • Mitochondrium: Centrum energetyczne (oddychanie kom.).
  • Chloroplast: Tylko rośliny, fotosynteza.
  • Rybosomy: Synteza białek.
  • Siateczka śródp.: Transport wewnątrzkomórkowy.

Genetyka

  • Genotyp: Zespół genów organizmu.
  • Fenotyp: Cechy widoczne (genotyp + środowisko).
  • Homozygota: AA (dominująca) lub aa (recesywna).
  • Heterozygota: Aa.
  • Kariotyp: Kompletny zestaw chromosomów (człowiek: 46).

Ekologia & Procesy

  • Fotosynteza: H2O + CO2 + Światło -> Glukoza + O2
  • Oddychanie: Glukoza + O2 -> ATP + CO2 + H2O
  • Mutualizm: Obie strony korzystają (obligatoryjny).
  • Komensalizm: Jeden korzysta, drugi nie traci.

Anatomia Człowieka

  • Krwionośny: Serce -> Tętnice -> Włosowate -> Żyły.
  • Oddechowy: Jama nosowa -> Krtań -> Tchawica -> Oskrzela -> Płuca.
  • Nerwowy: OUN (mózgowie + rdzeń) i obwodowy.
